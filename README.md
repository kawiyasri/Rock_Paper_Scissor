# 🎮 Rock Paper Scissors Game using Python (Tkinter)

A modern and interactive **Rock Paper Scissors Game** developed using **Python** and **Tkinter**. This project features a colorful graphical user interface, score tracking, configurable game rounds, restart functionality, and an engaging gameplay experience against the computer.

---

## 📌 Features

- 🎨 Interactive GUI built using Tkinter
- ✊ Rock, 📄 Paper, ✂️ Scissors gameplay
- 🤖 Random computer opponent
- 🏆 Real-time score tracking
- 🔢 Custom number of rounds
- ▶️ Start Game button
- 🔄 Restart Game button
- 📊 Round counter
- 🎯 Automatic winner announcement
- 🖼️ Image-based buttons and game display
- 🚪 Quit button to exit the game

---

## 🚀 Getting Started

### Prerequisites

- Python 3.8 or above
- Tkinter (comes pre-installed with most Python installations)

Verify installation:

```bash
python --version
```

Check Tkinter:

```bash
python -m tkinter
```

---

## 📥 Installation

Clone the repository

```bash
git clone https://github.com/your-username/RockPaperScissor.git
```

Navigate to the project directory

```bash
cd RockPaperScissor
```

Run the application

```bash
python rock_paper_scissor.py
```

---

## 🎮 How to Play

1. Launch the application.
2. Enter the number of rounds.
3. Click **Start Game**.
4. Choose one of:
   - Rock
   - Paper
   - Scissors
5. The computer randomly selects its move.
6. Scores are updated after every round.
7. Once all rounds are completed, the overall winner is announced.
8. Click **Restart Game** to play again.

---

## 📖 Game Rules

| Player Choice | Computer Choice | Result |
|--------------|----------------|--------|
| Rock | Scissors | Player Wins |
| Rock | Paper | Computer Wins |
| Paper | Rock | Player Wins |
| Paper | Scissors | Computer Wins |
| Scissors | Paper | Player Wins |
| Scissors | Rock | Computer Wins |
| Same Choice | Same Choice | Tie |
